# Everand EPUB Studio 1.0.0

Primeira versão pública do aplicativo Windows para reconstrução local de ebooks offline autorizados do Everand no LDPlayer.

## Destaques

- Interface gráfica nativa em português.
- Coleta automática por ADB do LDPlayer 9/14.
- Snapshots seguros e não destrutivos.
- Verificação de acesso integral e permissão de download.
- Reconstrução EPUB 3 com formatação, imagens, tabelas, fontes, links e navegação.
- Validação estrutural integrada.
- Distribuição `Trusted` compatível com Smart App Control.

## Download recomendado

Baixe `EverandEPUBStudio-Trusted.zip`, extraia todo o conteúdo e execute `EverandEPUBStudio.exe` dentro da pasta extraída.

O arquivo `.exe` também acompanha a release como referência, mas não é autônomo: ele precisa das pastas presentes no ZIP.

## Verificação

- Teste real no LDPlayer 14 com Android 14.
- Coleta e conversão ponta a ponta executadas pelo aplicativo empacotado.
- EPUBCheck 5.3.0 sem erros ou avisos.
- 96 componentes nativos com assinatura Authenticode válida.

## Escopo

Esta versão suporta ebooks baixados integralmente para leitura offline. Audiobooks não são suportados.

Use apenas com conteúdo cujo download e acesso tenham sido autorizados para sua conta.
